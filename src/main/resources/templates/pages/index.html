Bem-vindo à Jotta's Burger

Um sistema pensado para demonstrar autenticação, cardápio, carrinho, pedidos e organização de uma hamburgueria moderna.

Entrar no sistema Ver API

O que você encontra aqui

  • Login com área autenticada
  • Cardápio organizado por categorias
  • Carrinho de compras simples
  • Criação e acompanhamento de pedidos

Cardápio organizado

Navegue por categorias e produtos ativos de forma simples, clara e com foco na experiência do usuário.

Fluxo real de pedidos

O sistema permite simular um fluxo completo, desde login até montagem do carrinho e criação do pedido.

Backend robusto

Aplicação construída com Spring Boot, JPA, Spring Security, Thymeleaf e integração com a API documentada via OpenAPI.
